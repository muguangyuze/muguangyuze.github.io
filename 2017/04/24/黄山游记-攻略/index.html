黄山游记+攻略

发表于 2017-04-24 | |

南京–>黄山 自由行

  • 出发地：
    • 南京市
  • 目的地：
    • 黄山风景区
    • 宏村
  • 自由行时间：
    • 2017/4/19~2017/4/22
  • 交通路线：
    • 南京市318路公家做到南京站西——火车站后面就是小红山客运站
    • 南京小红山客运站~黄山景区 （直达黄山景区——山下汤口镇）
    • 汤口镇到达黄山换乘中心（即黄山游客集散中心），坐黄山景区大巴，20分钟左右，到达黄山景区云谷寺入口
    • 进入景区，全程步行爬山，下山从慈云阁，依然乘坐黄山景区大巴，到达黄山游客集散中心
    • 黄山景区~宏村，在黄山游客集散中心拼车前往宏村
    • 宏村~黄山屯溪，拼车
    • 黄山~南京，黄山客运总站
  • 住宿：
    • 2017/4/19晚，汤口镇云海楼宾馆（自我感觉很烂）
    • 2017/4/20晚，黄山景区排云楼宾馆，床位房
    • 2017/4/21晚，宏村景区仙客居客栈（很好的地方）
  • 相关景点：

相关攻略（包括坐车、食宿、游玩路线、天气、拍照等）

  • 坐车：
    • 南京出发：小红山站出发（车票98元/人，时间4.5个小时）直达黄山景区，方便快捷；
    • 汤口镇到黄山换乘中心，汤口镇住宿一般都提供免费接送服务；
    • 黄山景区大巴，19元/人*2（往返），历时15~20分钟到达入口；
    • 云谷索道，80元/人/单程；玉屏索道，90元/人/单程；西海地轨缆车，100元/人/单程；太平索道，80元/人/单程；
    • 黄山换成中心到宏村
      • 黄山~宏村专线：最晚下午4:30，票价20元/人
      • 徽黄旅游咨询中心：拼车（商务车），30元/人
    • 宏村到黄山市区（屯溪老街）
      • 客车专线：一小时一辆，20元/人
      • 拼车：随时都有，30元/人
  • 食宿：
    • 允许的话尽量出发前多买富含能量的食物，如肉干等
    • 汤口镇住宿一般90~100左右已经很好了，吃的尽量在入住的酒店吃就ok，单人30左右
    • 若担心出发时购买的食物不够，可以在汤口镇采购，尽量在酒店里买，比专门的超市便宜很多
    • 黄山上吃的东西基本都是正常的2倍价格，且份量极少
    • 若希望看日落日出，黄山山顶住宿需要提前预定，自由行建议购买独立床位房，140/元/晚，推荐首选白云宾馆（距离光明顶较近）、排云楼宾馆（距离丹霞峰较近）
    • 宏村，尽量住村里，方便吃饭，推荐仙居客客栈
    • 宏村吃饭尽量饭店吃，碰到仙客居这样的老板会给你便宜很多，特色毛豆腐、梅干菜鸭腿、黄山烧饼
  • 游玩路线：
    • 黄山景区路线地图
      • 推荐路线：后山上（云谷寺上），前山下（慈光阁下）
      • 若坐缆车，线路无所谓
    • 黄山景区–> 宏村–> 卢村，塔川 –> 西递 –> 屯溪老街
  • 天气
    • 天气选择雨后初晴的天气，一来不热，二来可以看奇观——云海及佛光、日出及落日
    • 注意防晒设施，带防晒霜和穿防晒服

  • 拍照

    • 黄山处处是美景，尽量带相机，不过手机也可以

  • 其他

    • 尽量轻装上阵，爬一爬山，体验下天都峰的刺激
