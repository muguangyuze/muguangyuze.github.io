被困电梯如何自救

发表于 2017-04-19 | |

今天遭遇了由于停电造成被困电梯的境况，觉得还是有必要写写被困电梯如何自救的。

基本情况

  • 晚间7：30分左右，刚到达黄山景区汤山镇的入住地点，办理入住、放好东西，准备下楼吃晚餐，突然电梯卡壳。
  • 由于不知道什么情况，刚开始信号全无，用手敲击电梯门，取得了酒店人员的注意。

  • 后靠近电梯门，讯号出现，但较弱，就拨打了求救电话，5分钟之后获救。

  • 电梯的内门还是很容易打开的，但是外门看酒店救援人员的情况，确实是很难打开的

注意事项

  • 一定要保持镇静，寻找求救办法，如敲击电梯（但也不要太过用力导致电梯摆动）、寻找求救电话
